# Layer 1: Structural DNA

How they build sentences, paragraphs, and use punctuation.

## Analysis Points

### 1. Sentence Length Patterns
- **Short:** 5-10 words
- **Medium:** 10-20 words
- **Long:** 20+ words
- **Variety score:** Mix vs consistent

Example output: "Short punchy sentences. One idea per line. Never rambling."

### 2. Paragraph Structure
- Paragraph length preference (1-liner vs chunky blocks)
- Opening sentence patterns (question, statement, transition)
- Closing sentence patterns (CTA, cliffhanger, summary)

### 3. Punctuation Signature
Analyze frequency of:
- Long dashes (the long horizontal line between words, like this)
- Ellipses (...)
- Question marks
- Parenthetical asides (like this)
- *Italics* for emphasis
- **Bold** for impact
- Exclamation points!
- Semicolons; colons:

### 4. List/Format Preferences
- Numbered lists vs bullet points vs no lists
- Headers/subheaders style
- White space usage
- Indentation patterns

## Example Output

```
LAYER 1: STRUCTURAL DNA

Sentence length: Short, punchy (avg 8-12 words)
Paragraphs: 1-3 sentences max, heavy white space
Punctuation: Frequent long dashes, occasional ellipses for pause
Lists: Bullet points preferred, never numbered
Formatting: Bold for key points, italics rare
```

### 5. Transition Vocabulary
How ideas connect between paragraphs and segments:
- **Bridge phrases:** What words/phrases start new paragraphs? ("So," "Okay, so..." "Here's the thing" "Now," "But here's where it gets interesting")
- **Topic shift markers:** How do they signal "new topic"? (explicit labels, natural flow, abrupt cuts)
- **Connective tissue:** What holds a long piece together? (callbacks to earlier points, running metaphors, numbered progression)
- **Tangent handling:** How do they return from tangents? ("So anyways," "But back to..." "The point is...")

Calculate: What % of paragraph transitions use explicit bridge phrases vs implicit flow?

Example output:
```
Transitions: 70% explicit bridges ("Okay, so..." / "Here's the thing"), 30% implicit flow
Topic shifts: Verbal marker ("Okay") then launch phrase ("So")
Tangent recovery: "So anyways" (casual redirect, no apology)
Connective tissue: Callbacks to opening hook, numbered walkthrough for tutorials
```

### 6. Closing Strategies Per Platform
How they end content across different formats:
- **Video:** What's the last 30 seconds? (CTA, teaser, homework, callback to hook?)
- **LinkedIn:** Last line pattern (question, one-liner, CTA, mic drop?)
- **X/Twitter:** Thread closers (summary, retweet bait, reply prompt?)
- **Email:** Sign-off pattern (personal, professional, CTA-heavy?)
- **YouTube Short:** Closing hook (loop back to start, punchline, cliffhanger?)

Note which platforms get formal closes vs casual trail-offs.

Example output:
```
Video: Value tease ("stick around") + homework + simple CTA
LinkedIn: One-sentence takeaway + question to audience
X: Summary of key point + "What's yours?" engagement prompt
Email: Personal sign-off with one link
Short: Punchline or loop back to hook
```

## Analysis Prompt

When analyzing samples, look for:
1. Count average sentence length across multiple pieces
2. Note paragraph breaks, where do they typically break?
3. Track punctuation frequency, what's overused vs avoided?
4. Identify formatting consistency
5. Map transition vocabulary and bridge phrase patterns
6. Compare closing strategies across platforms
